# PyTaskGantt Project Context

## Project Overview

**PyTaskGantt** is a Python-based tool for visualizing task schedules as Gantt charts. It offers two primary modes of operation:
1.  **Interactive Web App:** A Streamlit-based dashboard allowing real-time viewing and editing of task data.
2.  **Static Generation:** A script to generate a standalone HTML Gantt chart without the web interface.

The project uses **Plotly** for rendering interactive charts and **Pandas** for data manipulation.

## Key Files

*   **`create_gantt.py`**: The main entry point for the interactive Streamlit application. It loads data from `tasks.csv`, displays an editable table, and renders a reactive Gantt chart.
*   **`create_gantt_noStreamlit.py`**: A standalone script that reads `tasks.csv` and generates a static `Tasks.html` file containing the Gantt chart. Useful for automated reporting or when a web server isn't needed.
*   **`tasks.csv`**: The data source file. Stores task information including name, start time, end time, and assigned bot/resource.
*   **`xlsx2csv.py`**: A utility script to convert an Excel file (`xlsxname.xlsx`) to the required `tasks.csv` format.
*   **`Tasks.html`**: The output file generated by `create_gantt_noStreamlit.py`.

## Data Format (`tasks.csv`)

The system relies on a CSV file with the following columns:
*   **`Task`**: Name or identifier of the task.
*   **`Start`**: Start timestamp (format: `YYYY-MM-DD HH:MM:SS` or `HH:MM:SS` if same day).
*   **`Finish`**: End timestamp.
*   **`Bot`**: The resource, bot, or category assigned to the task (determines color coding).

Example:
```csv
Task,Start,Finish,Bot
Data Sync,2025-10-22 09:00:00,2025-10-22 09:25:00,Bot A
```

## Development & Usage

### Prerequisites
*   Python 3.7+
*   Dependencies: `streamlit`, `pandas`, `plotly`, `openpyxl` (for Excel conversion)

### Running the Interactive App
```bash
streamlit run create_gantt.py
```
*   Access the interface at `http://localhost:8501`.
*   Changes made in the UI can be saved back to `tasks.csv`.

### Generating Static Charts
```bash
python create_gantt_noStreamlit.py
```
*   Outputs to `Tasks.html`.

### Data Conversion
To convert an Excel file named `xlsxname.xlsx` to `tasks.csv`:
```bash
python xlsx2csv.py
```

## Conventions
*   **Date Parsing:** The scripts use `pd.to_datetime` with `errors='coerce'`, meaning invalid dates will result in NaT (Not a Time) and those rows may be filtered out.
*   **Chart Layout:** The Y-axis is fixed (does not zoom), while the X-axis (time) allows zooming and panning.
*   **Formatting:** Time is typically formatted as `%H:%M:%S` on the axis ticks.
